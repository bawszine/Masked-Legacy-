# Masked Legacy

**Masked Legacy** is a 2D top-down action-adventure game developed in Unity. The game focuses on fast-paced combat, skill-based movement, enemy AI, interactive objectives, and map progression.

The player explores different maps, fights enemies, completes specific tasks, unlocks gates, and moves forward through the game world. The gameplay includes sword combat, dash movement, healing abilities, boss fights, and interactive systems that make each level more challenging.

This project was developed as part of our game development journey and participated in the **Game Development Competition at Zarqa University**.

## Main Features

* 2D top-down player movement
* Sword-based melee combat system
* Dash and healing abilities
* Enemy AI with chase and attack behavior
* Boss fight mechanics
* Interactive tasks and progression system
* Multiple maps with different environments
* UI, audio, and visual effects
* Built using Unity and URP 2D

## Gameplay Loop

Enter the map → Fight enemies → Complete the objective → Unlock the gate → Move to the next map

## Technologies Used

* Unity
* C#
* Universal Render Pipeline 2D
* Tilemap System
* Rigidbody 2D
* Animator
* UI System

## Project Status

The core gameplay systems are completed, including player movement, combat, enemy AI, and basic progression. The game is still being improved through map polishing, bug fixing, UI improvements, audio, and final testing.
