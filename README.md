# 🎭 Masked Legacy - 2D Top-Down Action Adventure Game# 🎭 Masked Legacy - 2D Top-Down Action Adventure Game

## 🎮 About The Game

**Masked Legacy** is a 2D top-down action-adventure game built with **Unity** and **C#**.

The player explores different maps, fights enemies, completes objectives, unlocks gates, and progresses through the game world.

The game focuses on fast-paced sword combat, skill-based movement, dash and healing abilities, enemy AI, boss fights, and interactive tasks.

This project participated in the **Game Development Competition at Zarqa University**.

---

## 🕹️ Controls

The game supports both **keyboard & mouse controls** and **controller input**, allowing players to choose the control style they prefer.

### 🎮 Controller Support

* **Left Stick** → Move
* **L2** → Dash
* **R2** → Attack
* **Square** → Heal
* **X** → Talk / Interact / Skip Text

### ⌨️ Keyboard & Mouse

* **WASD** → Move
* **Left Mouse Click** → Attack
* **Shift** → Dash
* **H** → Heal
* **E** → Interact / Talk
* **Space** → Skip Text
* **Esc** → Pause Menu

## 🌍 Game World

**Masked Legacy** contains multiple maps with different environments and challenges.

Each map includes objectives that the player must complete before moving forward. Some areas are locked until the player defeats enemies, completes tasks, or activates interactive objects.

The world is designed using Unity Tilemaps, with obstacles, collision systems, gates, portals, and interactive elements.

---

## ⚔️ Combat System

The combat system is based on close-range sword attacks.

The player can attack enemies using melee combat, avoid danger using dash movement, and heal when needed. Enemies use AI behavior to chase the player and attack when close.

The game also includes boss fight mechanics to create more challenging combat encounters.

---

## ❤️ Health System

The game includes a health system for both the player and enemies.

The player can take damage from enemies and recover health using the healing ability. Health bars and UI elements are used to show the current health status.
